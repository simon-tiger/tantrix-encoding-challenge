Tantrix Tile Encoding Challenge

I'm trying to make Tantrix, but I need the tiles for it. And I would get tired if I would just make them myself. So I thought, if I ask other people to do it for me, then I would be able to get to the coding straight away without me making the dataset!

Format

So the 1st and 2nd position are in one question, and the color in the other one. These questions are there for each link. The layering order of the links is asked in the last question. If the order isn't selected, the order is simply 1 2 3.
